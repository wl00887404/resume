嗨！我是何振志！

2018 畢業於國立中正大學資管系，約有四年網頁開發經驗，有強烈的教學熱誠、樂於分享，積極參與技術社群，累積了大量的 code review 及與新人溝通的經驗，沉浸於幫助人與解決問題的成就感。

求學歷程

在高中時，電腦接觸了樂高機器人，我發現我對程式設計充滿興趣，我總是能很快的完成課程目標，因此大學選了資管系，我的學習進度一直比同學還快，對程式、心理、哲學相關的課程很特別有興趣，大學四年寫過無數種語言， ARM Assembly 、 C++ 、 Java 、 Python 、 Haskell 都多少接觸過，自選普通心理學、認知心理學。

大一時我開始嘗試接案，第一個案子是教授的產學合作案，專案需求是一個很複雜的 web 表單，教授推薦我以 ASP.NET Web Forms 、 Jquery 實作，雖然成品有達成專案需求，但是程式碼邏輯相當混亂，我想學校教學內容並不足讓我面對挑戰，在學長的建議之下，我開始加入程式設計的社群、參加 conference 。

在一次 Frontend 的讀書會，我第一次接觸 React，與我第一次接案的程式碼形成強烈的對比，React 清晰的邏輯令我相當震驚，這麼好的框架，不能只有我知道，大三我主動應徵當網頁程式設計的助教，利用暑期找時間辦讀書會、實作營，將自己學的新技術推廣給學弟妹，此時我也開始嘗試在線上讀書會開 webinar 分享 React Apollo ，我發現當我在準備課程內容時，我也更內化、熟悉這項技術，三年不斷的分享持續學習新技術、磨練教學技巧。

工作經歷

大四在經學長介紹在 Tmot 實習，主要職責是 React 前端工程師，但我很樂意處理其他技術問題，也寫過影音轉檔、後端伺服器、 SQL ，過去大量的教學經驗，讓我可以很快速的做 code review ，很順暢的與人溝通，在 Tmot 接觸了各式各樣 case ，每個月都有不同的成長，畢業後也在 Tmot 繼續轉正職。

在 Tmot 期間碰巧看到 AlphaCamp 在徵線上助教，僅僅需要在線上回復學生的問題、批改作業，是份很開心、不會造成多餘負擔工作，每當收到學生感謝的留言，都覺得開心不已。

因為兵役問題離開 Tmot ，退伍後經 Tmot 主管推薦去了 Canner 工作， Canner 主要產品為以分散式資料處理引擎 Presto ，我發現自己對 Canner 的產品並不是特別有熱情，產品的技術瓶頸並不是我擅長的技術領域，每天的工作都很單一，越做越倦怠，決定離開 Canner 後，稍微休息一下，再重新出發。

特殊成就

  • 在 AlphaCamp 時， 榮獲 2019 Q&A 回覆最多助教
  • 在學時， 榮獲 106 學年度優良助教
